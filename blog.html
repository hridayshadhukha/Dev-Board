Question-1: What are the different ways to select an element in the DOM?

DOM এ যেকোনো একটি এলিমেন্ট সিলেক্ট করার জন্য বেশ কয়েকটি ম্যাথড রয়েছে।
১। getElementById মেথডে id যুক্ত করে এটি সিলেক্ট করা যায়।
২।getElementByClass মেথডে class যুক্ত করে এটি সিলেক্ট করা যায়।
৩।getElementsByTagName মেথডে tagname যুক্ত করে এটি সিলেক্ট করা যায়। এছাড়াও querySelector() ও querySelectorAll() ব্যবহার করেও DOM এ Element select করা যায়।


Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML ট্যাগসহ টেক্সট রিটার্ন বা সেট করতে ব্যবহার করা হয়।
innerText শুধুমাত্র দৃশ্যমান টেক্সট রিটার্ন বা সেট করে।
textContent সম্পূর্ণ টেক্সট রিটার্ন বা সেট করে, এমনকি display: none; থাকলেও।


Question-3: What is event delegation in the DOM?

event delegation হল একটি প্রযুক্তি যেখানে একাধিক চাইল্ড এলিমেন্টের জন্য আলাদাভাবে ইভেন্ট লিসেনার যুক্ত না করে, তাদের parent এলিমেন্টে একটি ইভেন্ট লিসেনার সেট করা যায়। যখন কোনো চাইল্ড এলিমেন্টে ইভেন্ট ট্রিগার হয়, তখন সেই ইভেন্টটি বাবলিং (bubbling) এর মাধ্যমে প্যারেন্ট এলিমেন্টে পৌঁছে যায়, এবং আমরা সেই ইভেন্টটি হ্যান্ডেল করতে পারি।


Question-4: What is event bubbling in the DOM?

event bubbling হল একটি DOM ইভেন্ট প্রপাগেশন পদ্ধতি, যেখানে কোনো চাইল্ড এলিমেন্টে ইভেন্ট ঘটলে, সেটি প্রথমে সেই চাইল্ড এলিমেন্টের ইভেন্ট হ্যান্ডলার চালায় এবং তারপর ধীরে ধীরে প্যারেন্ট এলিমেন্টগুলোর মাধ্যমে উপরে উঠে।


Question-5: How do you create, add, and remove elements using JavaScript?

createElement মেথড ব্যবহার করা হয় যেকোনো ধরনের HTML এলিমেন্ট তৈরি করতে।যেমন div, span, p ইত্যাদি।

এলিমেন্ট DOM-এ যোগ করার জন্য appendChildমেথড ব্যবহার করা হয়, যা নতুন এলিমেন্টটিকে প্যারেন্ট এলিমেন্টের শেষের দিকে যোগ করে। এছাড়া, prepend() মেথড ব্যবহার করে এলিমেন্টটি প্যারেন্টের প্রথম দিকে যোগ করা যায়।

যখন এলিমেন্টটি আর প্রয়োজন হয়না, তখন আপনি remove() অথবা removeChild() মেথড ব্যবহার করে সেটি DOM থেকে মুছে ফেলতে পারেন।এই পদ্ধতি ব্যবহার করে আপনি ডাইনামিকভাবে এলিমেন্ট তৈরি, যোগ এবং মুছে ফেলতে পারেন।
